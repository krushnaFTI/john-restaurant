This extension for the Restaurant Menu by MotoPress allows your customers to add any available toppings for all presented menu items. The toppings are listed and available for purchase along with the Add to cart option. 
 
== Description ==

If you sell food and/or drinks that are also available with additional toppings (e.g. for pizza, salads, drinks, soups, etc.), this extension lets you output all of them right with each menu item featured with a help of your MotoPress Restaurant Menu plugin. Within the admin area you are free to add all needed toppings and tie them to the needed menu items. You can configure the following settings for a singular topping:


Add its name and description 
Define selection controls on the frontend for the users (checkbox, radio button, stepper) 
Set a price for each topping 
Add a featured image for a topping 
Switch on the option of changing the quantity of toppings on the checkout page
Change label for Add a topping button 
Edit description for Toppings option

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Restaurant Menu>Settings->Extensions>Toppings Settings screen to configure the plugin.
4. Use the Restaurant Menu>Toppings screen to add and configure your toppings.
