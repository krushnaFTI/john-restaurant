This extension for the Restaurant Menu by MotoPress enables delivery services that allow to optionally set delivery and pickup options.   

== Description ==

This plugin helps you sell food and drinks online and deliver them with one of the chosen options: delivery and pickup. The plugin comes packaged with an ultimate set of options for a proper use. You are free to set up the plugin for your needs using the following options: 


Turn on/off delivery option 
Define a price for delivery
Enable taxes 
Set a minimum order amount for a free delivery
Turn on/off pickup option
Edit the information about pickup to show to the customer 
Enable the feature that allows customers to choose time of delivery/pickup 
Define a minimum time interval from the time of purchase 
Preview and manage all orders with a handy admin interface  

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Restaurant Menu>Settings->Extensions>Delivery Settings screen to configure the plugin.
4. Use the Restaurant Menu>Order screen to configure the orders.
